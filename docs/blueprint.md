# **App Name**: Pookie Plays

## Core Features:

- Interactive Game Board: A 3x3 grid UI where players can make moves by clicking on cells, styled with rounded corners and soft shadows.
- Optimal AI Player (Minimax): An AI opponent utilizing the Minimax algorithm to ensure optimal gameplay for 'O', featuring a small 'thinking' icon animation during its turn to indicate AI processing.
- Game State Management: Logic to manage turns, detect win conditions (Player X, AI O), and declare draws.
- Real-time Game Status: Dynamic display of current game status, including whose turn it is ('Your Turn', 'AI Thinking...'), win/loss, or draw messages.
- Game Reset Functionality: A prominent reset button to quickly restart the game at any point.
- Move & Hover Animations: Subtle animations for cell hover effects and visual feedback when a move (X or O) is placed on the board.
- AI Companion Commentary: The AI provides lighthearted, encouraging, or humorous comments based on game events (e.g., player's move, AI's move, game outcome), using its reasoning as a tool to decide relevant output for fun interactions.

## Style Guidelines:

- Primary color: Soft lilac (#BD9FEA) to convey a playful and friendly atmosphere. Chosen to fit a 'cute' aesthetic, moving beyond typical bright purples.
- Background color: Very light desaturated lilac (#F9F7FB) providing a clean and soft canvas for the playful elements.
- Accent colors: A gentle pastel blue (#99B3FF) for highlights and calls to action, and a soft blush pink (#FFC0CB) for secondary interactive elements and to enhance the cute and colorful feel.
- Headline and body font: 'Poppins' (sans-serif) for its contemporary, geometric, and precise appearance, which complements the cute and modern aesthetic perfectly for short game status messages.
- Custom 'X' and 'O' markers with a slightly rounded or bubbly style to match the overall cute and playful theme. A small, animated 'thinking' icon (e.g., a stylized brain or lightbulb) will appear near the AI's area during its turn.
- A centrally aligned, responsive layout for both desktop and mobile, characterized by generously rounded corners on game elements and subtle, soft drop shadows for depth and coziness, reminiscent of a 'pookie' aesthetic.
- Smooth hover animations on board cells to indicate interactivity and small, subtle bounce or fade-in animations when X's and O's are placed to provide satisfying visual feedback. A distinct 'thinking' animation for the AI icon when it's processing its move.
